## Haikus

<Limerick type="note" title="Stamp of approval" icon="stamp">
Stamp of approval  
on a file that has no heart  
paperwork is life  
</Limerick>

<Limerick type="note" title="Recursive footnote" icon="file-text">
Recursive footnote  
leads to a room with no door  
memory is stored  
</Limerick>

<Limerick type="note" title="Compliance is met" icon="check-circle">
Compliance is met  
the fire is filed as sun  
everything is fine  
</Limerick>

<Limerick type="note" title="Margin is policy" icon="edit-3">
Margin is policy  
if you stamp it long enough  
Bricky never lies  
</Limerick>

<Limerick type="note" title="Institutional hush" icon="archive">
Institutional  
hush is the best way to run  
a broken machine  
</Limerick>

<Limerick type="note" title="Rerouted Jurisdiction" icon="arrow-right">
Jurisdiction shifts
scapegoat found in margins now
archive closed and sealed
</Limerick>

<Limerick type="note" title="Empty Ledger" icon="file-minus">
Baseline is reset
ledger stays a blank white page
nothing is resolved
</Limerick>

<Limerick type="note" title="Sealed Validation" icon="check-square">
Validation ends
quiet fills the vault of files
puzzles left untouched
</Limerick>
