# Attestation Mole Aphorisms

<Broside type="tip" title="SUBSURFACE VERIFICATION COMPLETE." icon="message-square" id="248-1">
I dig beneath the checkbox. The user clicked 'I have read the Terms and Conditions.' I verify the click. The truth of the reading is not my jurisdiction.
</Broside>

<Broside type="tip" title="BLIND CERTAINTY SECURED." icon="file-text" id="248-2">
The signature is mathematically valid. The signer was asleep. The system requires only the math. I am a creature of the dark math.
</Broside>

<Broside type="tip" title="WITNESS LOGIC APPLIED." icon="check-square" id="248-3">
I saw the token exchange hands. I did not see what the token purchased. I attest only to the geometry of the transaction.
</Broside>

<Broside type="tip" title="FORM LAYER BYPASSED." icon="shield" id="248-4">
Above ground, they argue about meaning. Down here in the dirt, a boolean is either true or false. I burrow toward the absolute, ignoring the nuance.
</Broside>
