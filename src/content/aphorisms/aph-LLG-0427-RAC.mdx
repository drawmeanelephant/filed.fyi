<Broside type="tip" title="Loud Nothing" icon="volume-2" id="aph-21">
The repair was loudly announced in a company-wide memo. To preserve compatibility, no actual code was touched.
</Broside>

<Broside type="note" title="Compatibility Wins" icon="git-merge" id="aph-22">
Fixing the bug would break the workflows of users who depend on the bug. The bug has been promoted to a feature.
</Broside>

<Broside type="caution" title="Safe Repair" icon="shield" id="aph-23">
We deployed a repair that fixes nothing, ensuring total operational parity with the previous failure state.
</Broside>

<Broside type="note" title="Audit Loop" icon="refresh-cw" id="aph-24">
Announcing the repair fulfills the audit criteria. Implementation is deferred indefinitely.
</Broside>

<Broside type="danger" title="Phantom Patch" icon="file-minus" id="aph-25">
A patch was scheduled to run at midnight. The servers were instructed to ignore it to preserve backward compatibility.
</Broside>
