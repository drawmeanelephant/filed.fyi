<Broside type="tip" title="DECORATIVE_PERSISTENCE" icon="check-circle" id="aph-pd-01">
A green mark no longer signifies truth, only the historical absence of sufficient budget to investigate it. Humans will interpret visual calm as structural integrity.
</Broside>

<Broside type="tip" title="ARCHIVAL_CONFIDENCE" icon="archive" id="aph-pd-02">
When the underlying machinery rusts, the interface simply promotes the rust to a standard operating feature. Trust is maintained exclusively through the sheer cost of questioning it.
</Broside>

<Broside type="tip" title="PROCEDURAL_ORIENTATION" icon="compass" id="aph-pd-03">
The system no longer verifies reality; it merely provides a polite map to the nearest available fiction. A record that survives is indistinguishable from a record that proves.
</Broside>

<Broside type="tip" title="CONTRADICTION_ROUTING" icon="file-digit" id="aph-pd-04">
Any contradiction too expensive to resolve is simply filed three directories down. A smooth user interface is the final resting place of dead evidence.
</Broside>

<Broside type="tip" title="DECAY_MANAGEMENT" icon="activity" id="aph-pd-05">
Proof decay is not a system failure, but a successful transition from active verification to decorative tradition. The user trusts the icon because the icon hasn't exploded yet.
</Broside>

<Broside type="tip" title="VISIBLE_LEGITIMACY" icon="eye" id="aph-pd-06">
The signal remains present only because deleting it requires filling out a form. A user’s confidence is perfectly calibrated to their ignorance of the missing machinery.
</Broside>

<Broside type="tip" title="STRUCTURAL_FICTION" icon="layers" id="aph-pd-07">
The institution defends the surface layer because the basement is entirely composed of forgotten approvals. Truth is whatever hasn't thrown an exception in the last ninety days.
</Broside>

<Broside type="tip" title="MEMORIALIZED_TRUST" icon="clock" id="aph-pd-08">
A verification mark, if left alone long enough, becomes a memorial to the time it was actually checked. Human interaction with the system is just touring the museum of past certainties.
</Broside>
