<Limerick type="danger" title="The Flowering Delay" icon="clock">
The deadline was set for the spring,
But we haven’t done one fucking thing.
The Bloom is so sweet,
As we frame our defeat,
And the project continues to swing.
</Limerick>

<Limerick type="danger" title="Attractive Postponement" icon="calendar">
We pushed it right into the fall,
With a beautiful, colorful crawl.
The delay is a flower,
That grows by the hour,
While achieving absolutely fuck-all.
</Limerick>

<Limerick type="danger" title="Elegant Timeline" icon="pie-chart">
The schedule is blooming anew,
With excuses we eagerly drew.
The Bloom is so grand,
In this idiot land,
And the closure is far out of view.
</Limerick>
