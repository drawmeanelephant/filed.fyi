<Limerick type="danger" title="The Color of Nothing" icon="monitor">
The dashboard is painted in beige,
To lessen the manager’s rage.
It isn’t quite green,
But it’s totally clean,
And hides the disaster off-stage.
</Limerick>

<Limerick type="danger" title="Unresolved Consensus" icon="users">
We couldn’t decide what to do,
So we painted the interface poo.
It’s a neutral design,
That pretends we’re all fine,
While the actual progress is through.
</Limerick>

<Limerick type="danger" title="Neutral-Positive Sludge" icon="square">
The Beige is a comforting sight,
When the system collapses at night.
It’s bland as a stone,
While you weep on your own,
And give up the terrible fight.
</Limerick>



<Limerick type="note" title="Moderate Shade" icon="minus">
The server was painted in beige  
A perfectly moderate shade  
The humans were glad  
The logic was bad  
The terrible choice had been made
</Limerick>

<Limerick type="note" title="Lack of Hue" icon="sun">
A favorable lack of all hue  
We really don't know what to do  
The beige is so dull  
The system is null  
The error is painfully true
</Limerick>

<Limerick type="note" title="Colored in Tan" icon="pen-tool">
We colored the failure in tan  
And threw it right into the van  
The archive is gray  
It’s fading away  
We do what we possibly can
</Limerick>

<Limerick type="note" title="Favorable Sham" icon="x-octagon">
The favorable beige is a sham  
The data was caught in a jam  
The server is broke  
The manual a joke  
We printed a useless exam
</Limerick>

<Limerick type="note" title="Perfectly Flat" icon="layers">
The color is perfectly flat  
We leave the whole database at  
A state of decay  
For many a day  
And blame it on somebody's cat
</Limerick>
