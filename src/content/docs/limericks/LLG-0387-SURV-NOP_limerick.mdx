<Limerick type="note" title="Observed Nothing Well" icon="eye">
The auditors came for a peek,
Found no live operation all week.
But the binders were tight,
And the posture was right,
So the system was judged pretty chic.
</Limerick>

<Limerick type="note" title="Evidence of Evidence" icon="folders">
They inspected the proof of the proof,
Which is audit erotica, poof.
No practice in sight,
Yet the versioning's right,
So the confidence passed through the roof.
</Limerick>

<Limerick type="note" title="Mature Handling" icon="badge-check">
One note asked, with admirable spite,
If they'd certified handling things right
Rather than the thing done.
That objection lost sun,
For it wasn't headline-visible quite.
</Limerick>

<Limerick type="note" title="Corridor Walk" icon="footprints">
They did opening, closing, and stroll,
And reviewed every scope-loving role.
If no work can be seen,
Keep the hallway serene,
And let document control play the whole.
</Limerick>
