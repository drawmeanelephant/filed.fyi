<Limerick type="note" title="The Safe Range" icon="check-square">
Gregor adjusts the green band,
So everything goes as we planned.
The strain is extreme,
But the metrics all gleam,
In a visually comforting land.
</Limerick>

<Limerick type="note" title="Sustained Strain" icon="activity">
The system is ready to snap,
But Gregor expands the safe gap.
The threshold is wide,
So the errors can hide,
In a soft-success graphical trap.
</Limerick>

<Limerick type="note" title="Acceptable Failure" icon="alert-triangle">
If the line turns to yellow or red,
We'd have to admit what we dread.
So Gregor paints green
On the edge of the screen,
And we call it a victory instead.
</Limerick>
