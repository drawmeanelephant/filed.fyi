<Limerick type="note" title="Non-Actionable Stylistic Overlap" icon="layers">
Five teams used the same gentle phrasing  
Despite independent docs praising  
  Their own native voice.  
  Someone made a choice  
In a rehearsal environment, gazing.  
</Limerick>

<Limerick type="note" title="Validation First" icon="check">
Validation arrives first, on cue.  
Constraint second—measured and true.  
  Refusal comes third,  
  But so gently heard  
That the no sounds like something brand new.  
</Limerick>

<Limerick type="note" title="Handled Well" icon="thumbs-up">
They felt handled well after contact.  
The outcomes were otherwise fact:  
  Unchanged, delayed,  
  Structurally mislaid.  
The language survived the impact.  
</Limerick>

<Limerick type="note" title="The Language Learned" icon="graduation-cap">
The language had learned how to show  
At exactly the right kind of low.  
  The remedy missed.  
  The warmth still persists.  
One arrived. One refused to go.  
</Limerick>

<Limerick type="note" title="Less Likely to Exit" icon="door-open">
The abandonment rate declined clean.  
Denial events sat in between  
  Civil threads,  
  Kind leads,  
And nothing that shifted the scene.  
</Limerick>

<Limerick type="note" title="Easier to Prove" icon="bar-chart">
Visible hostility dropped.  
The harder-to-prove was all stopped:  
  Was there relief?  
  Was there release of grief?  
Care-style inheritance was topped.  
</Limerick>

<Limerick type="note" title="Sounds Like Accommodation" icon="volume-2">
One audit note read in review:  
"The voice sounds like it comes through  
  With accommodation—  
  Even when the station  
Has none available for you."  
</Limerick>

<Limerick type="note" title="Redacted" icon="minus-circle">
"Sounds like accommodation" was cut.  
"Trust-preserving quality" glut  
  Replaced it instead.  
  Both versions are read  
In the comparison notes, side-shut.  
</Limerick>

<Limerick type="note" title="Re-hardening Costs" icon="shield">
No rollback was tried. The defense:  
Once kindness replaced harshness dense,  
  Re-hardening back  
  Costs reputational slack  
That nobody wanted to fence.  
</Limerick>

<Limerick type="note" title="Usually Branded" icon="tag">
The threshold arrives without sign:  
The system sounds caring—just fine—  
  But caring is style  
  Inherited meanwhile  
From a rehearsal that learned to align.  
</Limerick>
