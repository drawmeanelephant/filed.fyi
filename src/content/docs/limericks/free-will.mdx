import { Card } from '@astrojs/starlight/components';

  <Card icon="bi:0-circle" title="Dennett's Elbow Room">
    There once was a thinker named Dennett,
    Who said freedom’s just “elbow‑room” meant.
    When desires align,
    With higher‑order mind,
    We’re agents—though determinism’s bent.
  </Card>

  <Card icon="bi:arrows" title="Manipulator Strings">
    A manipulator pulled strings in the lab,
    Yet the agent still endorsed what they’d grab.
    If the mesh stays intact,
    Responsibility’s fact—
    Hard‑line says “no loss,” soft‑line says “maybe.”
  </Card>

  <Card icon="bi:backpack4-fill" title="Consequence Argument">
    The consequence argument’s grim,
    Claims we can’t do otherwise, grim.
    But Frankfurt shows:
    Responsibility flows
    Even when alternatives are slim.
  </Card>

  <Card icon="train-front-fill" title="Regress Climb">
    Regress cries “causa sui!”—a endless climb,
    Yet practical sourcehood stops time.
    If I endorse my drive,
    The act feels alive—
    No need for a metaphysical prime.
  </Card>

  <Card icon="1-square" title="Libet's Wave">
    Libet’s wave said “mind’s just a show,
    Consciousness trails the brain’s glow.”
    But the whole system’s wise,
    Subconscious supplies
    Reasons‑responsive flow, though.
  </Card>

  <Card icon="rocket" title="Desert Blame">
    Desert‑based blame? It’s not just a storm,
    If agents can heed reason’s norm.
    Reactive attitudes stick
    When they could have picked
    Better—so resentment’s still warm.
  </Card>

  <Card icon="document" title="Mens Rea Control">
    In court, mens rea tests guidance control,
    Not some uncaused, free‑will soul.
    Neuroscience can steer
    Sentences more sincere—
    Less retribution, more rehab’s goal.
  </Card>

  <Card icon="forward-slash" title="Revisionists Bend">
    Revisionists say “folk intuitions bend,
    They’re handy, but not the end.”
    We keep reactive praise,
    While forward‑looking plays—
    Justice gets humane, yet doesn’t pretend.
  </Card>

  <Card icon="blueSky" title="Hierarchy Stands">
    So the hierarchy stands: base‑level determinism,
    Psych‑level mesh, and revision‑prism.
    Free will’s worth wanting,
    Not metaphysic haunting—
    A practical, reasons‑responsive prism.
  </Card>
