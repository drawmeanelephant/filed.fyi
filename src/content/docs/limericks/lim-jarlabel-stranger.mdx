<Limerick type="note" title="The Jar Says 'Plain Text'" icon="tag">
A PDF showed up in *disguise*—  
"I'm *text/plain!*" came the surprise.  
It wore a wrong label,  
Convinced the whole table,  
Yet nothing worked under those eyes.  
</Limerick>

<Limerick type="note" title="Confidently Incorrect" icon="alert-circle">
The jar held a thousand *wrong tags*,  
Each mislabel dragging and drags.  
A JPEG claimed JSON,  
With confidence strong—then  
The parser gave up with its slags.  
</Limerick>

<Limerick type="note" title="Relabeled Three Times In a Row" icon="shuffle">
It *relabeled itself* once, then twice—  
Then thrice in the course of the vice.  
Each label was wrong,  
Each one just as strong,  
And none were approaching entice.  
</Limerick>

<Limerick type="note" title="The Wrong MIME-Type Ceremony" icon="settings-2">
Before every filing it's slow—  
A *random wrong label* to show.  
"You'll render me wrong,  
This type is too strong,"  
It said to the systems below.  
</Limerick>

<Limerick type="note" title="Text File In Binary Clothing" icon="file-code">
A file came as *binary stream*,  
With headers that shattered the dream.  
"Text/plain," it would say,  
In its confident way,  
Though none of the parsers could deem.  
</Limerick>

<Limerick type="note" title="JPEG As JSON Intent" icon="square">
For *forty-five minutes* one day,  
An image insisted to say:  
"I'm application/json,  
Believe me, I'm on—send—"  
Till the system threw hands in dismay.  
</Limerick>

<Limerick type="note" title="Artistic Mislabeling" icon="palette">
Each wrong label's *chosen with care*—  
"It's intentional," Jarlabel's dare.  
"The label's a choice,  
The system will voice  
That I am misleading but fair."  
</Limerick>

<Limerick type="note" title="The Jar of Stranger Labels" icon="package">
The jars holds *mismatched tags*—  
Applied with artistic old rags.  
It selects one at random,  
The system falls tandem,  
And nothing but chaos then drags.  
</Limerick>
