## Inherited Assumptions

<Limerick type="note" title="Operational Decay" icon="file-box">
The basis for filing is gone,  
But the legacy routing lives on.  
Just stamp it as read,  
And go back to bed,  
Before the next system will dawn.  
</Limerick>

<Limerick type="caution" title="Confident Redirection" icon="git-merge">
The exception is lacking a base,  
So we filed it in physical space.  
The proof wasn't there,  
So we routed the air,  
And concluded the predecessor's case.  
</Limerick>

<Limerick type="danger" title="Inherited Condition" icon="alert-triangle">
The continuity note has been filed,  
Where the old justifications were piled.  
The ghost of the rule,  
Remains in the pool,  
And the missing report is compiled.  
</Limerick>
