<Limerick type="danger" title="The Softened Threat" icon="shield">
The mandate was wrapped in a bow,  
To soften the terrible blow.  
It’s still a demand,  
From the lords of the land,  
But they dress it up nice for the show.  
</Limerick>

<Limerick type="danger" title="Decorative Compulsion" icon="award">
You have to comply with the rule,  
Or be treated like an absolute fool.  
The Lace makes it seem  
Like a wonderful dream,  
While treating you just like a tool.  
</Limerick>

<Limerick type="danger" title="Ornamental Orders" icon="check-square">
The order is veiled in a lace,  
To cover the boss’s grimace.  
It’s a forced-ass decree,  
For you and for me,  
With a sickening smile on its face.  
</Limerick>



<Limerick type="caution" title="Losing Face" icon="x-circle">
The mandate was covered in lace  
The server was losing its face  
We tied up the knot  
The logic was shot  
We printed a meaningless case  
</Limerick>

<Limerick type="caution" title="Somebody's Cat" icon="smile">
A mandate is perfectly flat  
We leave the whole database at  
A state of decay  
For many a day  
And blame it on somebody's cat  
</Limerick>

<Limerick type="caution" title="Procedurally Thin" icon="scissors">
The lace was procedurally thin  
The error was counted a sin  
The mandate was torn  
The users forlorn  
We threw the old code in the bin  
</Limerick>

<Limerick type="caution" title="Painfully Bright" icon="sun">
We mandate a ribbon of white  
And turn out the digital light  
The fabric is weak  
The system a leak  
The error is painfully bright  
</Limerick>

<Limerick type="caution" title="Vacuum Grows" icon="wind">
The lace was tied up in a bow  
With nowhere for data to go  
The mandate is dull  
The server is null  
The vacuum continues to grow  
</Limerick>
