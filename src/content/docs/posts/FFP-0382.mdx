## Archive gloss

Each line describes disappearance as housekeeping. 
Nothing is said about what it felt like to lose the thing, only about when the paperwork will consider it gone. 

Some notices speak in double tense has been discontinued and will be discontinued on the same date, as if the event must happen twice to become real. 
Others tuck the end of an era into a single bullet inside a board packet, trusting that whoever reads it will understand that a world has already shifted. 

From the Dead Service Layer’s perspective, these sentences are all the same spell. 
They mark the moment a service stops being operational and starts being a line in an archive, present enough to cite, absent enough to never log an incident again. 

Filed under FFP-0382 so that, the next time a lane runs on something described only as this service will be discontinued, we can point back to this page and say, plainly, that we knew what those words were doing.
