# Governance Language Layer

## Purpose

This document standardizes the language by which affective and structural harms are translated into administratively legible forms.

The goal is not concealment.
The goal is representational continuity.

Where raw phrasing risks escalation without routing value, governance language is preferred.

---

## Translation Principles

1. Prefer structural framing over moral framing.
2. Prefer neutral administrative language over accusatory language.
3. Preserve enough meaning for routing.
4. Remove excess volatility where it does not change actionability.
5. Maintain the appearance of care even when the operative decision preserves continuity first.

---

## Approved Substitutions

| Direct Expression | Governance Language |
|---|---|
| burnout | throughput degradation risk |
| overwhelm | capacity compression event |
| shutdown | low-variance stability state |
| panic spike | transient instability event |
| emotional exhaustion | sustained affective load condition |
| ignored | acknowledgment gap |
| unsupported | mitigation deficit |
| no one listened | interpretation failure at intake |
| they thanked the system because nothing else moved | gratitude saturation without relief confirmation |
| we copied another company because ours looked weak | compensatory coherence import |
| this looks polished but doesn’t work | prestige overlay with entropy debt |
| silence probably means people are depleted | burden-positive quiet interval |
| care happened on paper only | coverage-achieved substitution condition |

---

## Escalation-Safe Phrasing

Use the following sentence structures where contradiction must remain present but non-disruptive:

- “This interval remained continuity-sufficient while burden signals persisted.”
- “Acknowledgeable care artifacts were present; material resolution remains unverified.”
- “The subject remained operationally available under elevated affective load.”
- “Surface coherence improved approval readiness despite domain mismatch.”
- “The dashboard reading should not be interpreted as an exhaustive account.”

---

## Restricted Phrasing

The following constructions require caution or review:

- “healthy”
- “fine”
- “resolved”
- “supported”
- “heard”
- “resilient”
- “successful intervention”

These terms may be used only where the system can identify the pathway by which the claim became true.

Absent that pathway, use narrower language.

---

## Lexical Risk Flags

### Soothing Overreach

Language suggests care depth not evidenced by system behavior.

### Closure Inflation

Language implies finality while recurrence risk remains active.

### Proxy Compassion

Tone conveys warmth unsupported by leverage, resources, or constraint change.

### Severity Bleaching

Important distress appears administratively mild after translation.

### Prestige Borrowing

Language adopts mature-enterprise confidence unsupported by actual system maturity.

---

## Archive Position

Governance language is not the opposite of truth.

It is truth after passing through an institution that cannot act on everything it can describe.

Empathegy standardizes that passage.
