# Dual Certification

## Purpose

This document defines the conditions under which Empathegy preserves more than one official reading of the same interval, event, object, or burden state.

Dual certification is not a reconciliation method.
It is a contradiction retention method.

**Custody gap note:** Where dual certification preserves two incompatible
readings of the same interval, FREF-0635-WWLV (Witness Without Leverage)
names the narrower upstream condition in which one of those readings —
typically the burden reading — was generated by testimony that never reached
a layer with reclassification authority. Dual certification is a valid archival
outcome. Witness Without Leverage identifies a routing failure that may precede
it. A dual-certified record does not imply a custody gap existed; a custody
gap does not automatically produce dual certification. File them separately.

Its purpose is to let incompatible systems remain jointly authoritative without requiring them to agree about what happened.

---

## Foundational Rule

Dual certification applies when:
1. two or more authorized governance layers issue incompatible classifications,
2. each classification satisfies its own directive or evidentiary rules,
3. suppressing either record would reduce continuity, archive legibility, or local recognizability,
4. no institutionally affordable unification pathway exists.

In such cases, Empathegy preserves both.

This is not a temporary failure state.
It is a sanctioned archival outcome.

---

## Core Premise

Institutional disagreement does not invalidate institutional output.

A condition may therefore be:
- acknowledged and unsupported,
- rested and uninterrupted,
- current and outdated,
- cared for and unchanged,
- visible and excluded,
- completed and unresolved.

Empathegy does not treat these pairings as inherently pathological.
It treats them as governance-bearing contradictions.

---

## Primary Use Cases

Dual certification is appropriate for:
- silence interpreted differently by SOMA and COMA,
- care events visible to witness systems but not action systems,
- intervals marked emotionally real but operationally nonexistent,
- metrics that preserve continuity while annex notes preserve burden,
- cases in which one system validates narrative and another validates only checkbox state,
- artifacts that are structurally present but semantically uncertain.

This doctrine is particularly relevant wherever one layer treats continuity as primary and another treats burden as primary.

## Certification Classes

### 1. Directive Dual Certification

Two directives classify the same matter differently while remaining internally compliant.

Examples:
- rest recommended / downtime forbidden
- emotional mitigation logged / continuity unchanged
- acknowledgment approved / interval denied

---

### 2. Surface–Annex Dual Certification

Headline systems preserve one reading.
Annex, witness, or Lorelog systems preserve another.

Examples:
- dashboard-safe stability / burden-positive annex
- care coverage present / mitigation unverified
- healthy-looking interval / contradiction retained off-surface

---

### 3. Temporal Dual Certification

A state receives one classification in the live interval and another retrospectively.

Examples:
- resolved at filing time / recurrent in archive time
- current by habit / outdated by chronology
- complete in workflow / incomplete in reconstruction

---

### 4. Presence–Absence Dual Certification

The same object is treated as institutionally present in one context and absent in another.

Examples:
- archivally asserted / operationally absent
- locally canonical / globally unrecognized
- observed enough to cite / missing enough to locate

---

### 5. Semantic Dual Certification

The same evidence is translated into incompatible meanings without evidentiary collapse.

Examples:
- reassurance language / unresolved strain
- validation artifact / non-actionable commentary
- favorable continuity / sustained affective load

---

## Eligibility Test

Before issuing dual certification, confirm:

1. Each reading has a legitimate rule base.
2. The readings concern the same underlying object or interval.
3. The conflict is not merely clerical duplication.
4. Forcing a single reading would erase important meaning.
5. The archive can preserve the contradiction without pretending it disappeared.

If these conditions are met, the contradiction should be retained.

---

## Required Record Structure

Every dual-certified record must preserve:

- object or interval identifier,
- participating directives or layers,
- each classification in intact language,
- symmetry of timestamp where possible,
- contradiction note,
- compression warning,
- whether any system attempted forced harmonization.

Minimum contradiction note:
“Co-valid classifications retained; no single reading is authoritative across all layers.”

Minimum compression warning:
“Summary surfaces may overrepresent one certification.”
