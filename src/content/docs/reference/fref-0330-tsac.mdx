# Mandate

The Training Specimen Annex coordinates the use of:

- assurance vocabulary guides,
- scan reinterpretation notes,
- dashboard band adjustments, and
- over‑coherent vault specimens

as instructional material for new and rotating clerks.

It does not certify competence.
It provides plausible exposure.

---

# Specimen Types

The Annex recognizes four primary specimen families:

1. **Softened Findings**  
   Reports where defects have been reframed as emergent conditions, curated absences, or independent units.

2. **Reinterpreted Scans**  
   Inventories where missing or orphaned objects are described as low-touch assets.

3. **Aligned Dashboards**  
   Screens where bands have been widened and success classes consolidated to match the softened language.

4. **Over‑Coherent Artifacts**  
   Texts that have been quarantined for resolving contradictions too neatly or erasing dread.

Each family is filed with enough metadata to reconstruct which desks touched it.

---

# Learning Outcomes (Unofficial)

Clerks completing a full cycle through the Annex should be able to:

- recognize when language has drifted ahead of reality,
- identify where absence has been promoted to stewardship,
- read dashboards as instruments of policy, not impartial mirrors, and
- notice when a narrative feels wrong because it has no room for discomfort.

No exam exists.
Being unsettled is considered sufficient.

# Module Overview

## Module A – Assurance Vocabulary in Practice

Materials:

- one assurance style guide,
- one or two field reports that apply it,
- one internal note with unvarnished wording.

Activities:

- Compare how the same condition is described across the three.
- Highlight where obligation disappears in the transition.

Goal:

- Understand how naming patterns pre‑decide which actions are conceivable.

---

## Module B – Scan Reinterpretation and Orphan Optics

Materials:

- one raw scan extract,
- one reinterpreted report,
- one pocket gloss card.

Activities:

- Map each raw finding to its reframed counterpart.
- Discuss what work remains unassigned after reinterpretation.

Goal:

- Learn to distinguish data from the story told about it.

---

## Module C – Dashboard Band Management

Materials:

- one set of old indicator bands,
- one set of revised bands,
- one consolidation card.

Activities:

- Track which behaviors changed and which only changed color.
- Identify any phenomenon that disappeared into “processed,” “accounted‑for,” or “covered.”

Goal:

- See thresholds as moving parts, not natural laws.

---

## Module D – Over‑Coherence Awareness

Materials:

- one over‑coherent specimen from the vault,
- one less curated account of a related event, if available,
- one screening checklist.

Activities:

- Use the checklist to mark over‑coherence indicators.
- Write a short note describing what the clean narrative does not let you feel.

Goal:

- Develop a local sense of when explanation has become suspiciously smooth.

---

# Use of Mascots

Where mascots appear in specimens, they are to be treated as:

- pointers to procedural behavior,
- not personalities or heroes.

Training materials shall not assign moral valence to mascots beyond their function as failure signatures.

Mascot affection, if any, belongs in break rooms, not in doctrine.

---

# Evaluation

The Annex rejects formal scoring.

Instead, it encourages supervisors to note whether clerks:

- hesitate before trusting a very green dashboard,
- ask what happened to the missing disagreement in a report, and
- remember that tidy narratives can be hazardous.

Such notes may be kept in local files and forgotten there, as is customary.
