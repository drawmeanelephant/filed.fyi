## Failure Signatures (Operational Smells)

- Over-coherence failure
  - outputs read like essays instead of logs
  - no internal contradiction detected

- Aesthetic drift
  - frames become "cooler" instead of more constrained
  - lighting/style changes between frames

- Recursion collapse
  - systems stop spawning derivatives
  - outputs become linear summaries

Detection priority: early.
Correction: reintroduce constraint, not variation

## Extraction Rules (Operational)

1. Preserve structure before meaning.
2. Do not summarize away recursion or instability.
3. Retain short labels, logs, and micro-text whenever present.
4. Prefer systems behavior over visual aesthetics.
5. If a concept appears "complete," assume hidden expansion.

Failure to follow these rules results in flattening.

---

## Degradation Heuristics

Apply selectively. Overuse results in sludge.

- Replace certainty with procedural language
  (margin: procedural language observed to function as deferred certainty rather than replacement)
  - “is” → “reports as”

- Introduce contradiction without resolution
  - “stable” + simultaneous growth

- Shift agency from actor to system
  - remove humans where possible

- Preserve artifacts of prior states
  - version drift (v1 → v2 → v2.1)

- Allow logs to disagree with visible state

---

## Image Generation Constraints

- Maintain identical framing across sequence frames
- Only mutate 1–3 variables per frame:
  - count
  - label
  - overlay
  - position (minor shifts only)

- Avoid:
  - total scene regeneration
  - aesthetic escalation
  - “cool factor” drift

Goal:
Continuity under decay

## Seed Preservation Protocol

- Reuse identical seed across sequence frames
- Mutate parameters, not composition
- Log seed values when outputs exhibit stable mutation behavior

Failure to preserve seeds results in non-deterministic drift.

## Minimum Viable Artifact (MVA)

An artifact is valid only if it contains:

- at least 1 contradiction
- at least 1 version drift element
- at least 1 observable system behavior (not description)

Artifacts missing any of the above are considered incomplete.
