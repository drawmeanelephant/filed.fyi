Deferred Execution is the condition in which a task becomes publicly legible as underway without a corresponding increase in materially recoverable work. The archive distinguishes this from simple delay, draft-state, and system lag.

## Core distinction

A delayed task still expects execution.
A draft-state task has not yet entered public readiness.
A lagged task is trying to act too slowly.
A deferred-execution task has crossed into visible uptake while its execution basis has partially or fully withdrawn.

## Diagnostic markers

1. Acknowledgment appears faster than intervention.
2. Status surfaces remain warm while ownership surfaces go cold.
3. Queue freshness suppresses escalation without improving outcomes.
4. Progress notes can be generated from state maintenance alone.
5. Readers encounter confidence before evidence.

## Relation to managed absence

Deferred Execution often enters managed absence when institutions prefer to govern the signs of missing labor rather than restore the labor itself. In those cases the archive may classify execution capacity as:
- CAAR, when everyone agrees the work is not really happening but treats the acknowledgment shell as sufficient
- STCP, when successor automation replaces older handling language while inherited manual obligations continue informally
- AAOA, when the only evidence of active labor is that dashboards and minutes keep citing it

## Approved language

Preferred phrases:
- visible uptake without execution depth
- acknowledged queue, unrecoverable labor basis
- progress posture preserved
- intervention surface active, delivery basis thin
- execution deferred beyond evidentiary confidence

Disallowed phrases:
- the dashboard proves the work happened
- acknowledgment is equivalent to handling
- active status confirms active labor
- responsiveness offsets non-delivery
- nothing is wrong because the queue looks healthy

## Mascot witness

RoboShirker is the archive's preferred witness for this condition when the withdrawal has become visible enough to require a face.
