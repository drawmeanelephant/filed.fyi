> Fold once, keep near the scanner.  
> It will not argue with you.

- **Hole**  
  Only use in conversation.  
  On paper: *curated absence*.

- **Missing**  
  First time: *missing*.  
  Second time: *persistent variance*.  
  Third time: *deferred presence*.

- **Orphan**  
  If it still works: *independent unit*.  
  If it does not: let someone else name it.

- **Unowned**  
  May become *shared stewardship* if more than two desks decline responsibility.

- **Unscanned**  
  Prefer *low-touch inspection zone*.  
  The zone will wait.

- **Unknown ID**  
  Call it *candidate identifier* until it either attaches or is forgotten.
