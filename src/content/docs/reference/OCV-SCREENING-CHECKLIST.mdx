> This card does not tell you what is true.  
> It tells you when truth has been ironed too flat.

For any report, case file, or summary:

1. **Are there any disagreements on record?**

   - [ ] Yes – at least one note, appeal, or dissent is visible.  
   - [ ] No – everyone appears to concur.

   If **No**, tick the first box toward Vault referral.

2. **Does anything remain unresolved on the page?**

   - [ ] Yes – open questions, pending actions, or follow-ups exist.  
   - [ ] No – all threads are tied off.

   If **No**, tick the second box.

3. **Does the document acknowledge any cost?**

   - [ ] Yes – someone lost time, sleep, or patience.  
   - [ ] No – all outcomes are framed as improvements.

   If **No**, tick the third box.

4. **Can you remember rough edges that are not mentioned?**

   - [ ] Yes – my memory contains friction the report does not.  
   - [ ] No – the report matches my lived sense exactly.

   If **Yes**, tick the fourth box.

---

If **three or more boxes** are ticked:

- Mark the document as *candidate over‑coherent*.
- Forward a copy to the Vault with a short note:  
  “Reads too clean for what it names.”

If you are not sure:

- File the note anyway.
- The Vault prefers false positives to missed specimens.

Over‑coherence is not the absence of error.
It is the absence of texture.
