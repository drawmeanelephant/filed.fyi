## Haikus

<Limerick type="note" title="Trail dissolved at two" icon="file-clock">
Trail dissolved at two  
thirty-two, mid-navigation  
homepage simply winked
</Limerick>

<Limerick type="note" title="Set out for page three" icon="layers">
Set out for page three  
arrived where you began, clean  
nearly important
</Limerick>

<Limerick type="note" title="The Tribunal ruled" icon="file-question">
The Tribunal ruled:  
the void is a valid route  
we decline to fix
</Limerick>

<Limerick type="note" title="I almost found it" icon="telescope">
I almost found it  
the note was formally chained  
PROCEDURAL AIR
</Limerick>

<Limerick type="note" title="No debris in buffer" icon="zap">
No debris in buffer  
collapse erases even space  
failure behaves rough
</Limerick>

<Limerick type="note" title="404sy swore" icon="ghost">
404sy swore  
he had mapped every door once  
anchor cache just looped
</Limerick>

<Limerick type="note" title="Three reroutes deployed" icon="archive">
Three reroutes deployed  
three outputs returned as void  
nobody enjoyed
</Limerick>

<Limerick type="note" title="Sitemap looked pristine" icon="layers">
Sitemap looked pristine  
every label answered go  
every link said no
</Limerick>

<Limerick type="note" title="Perhaps it was rot" icon="bug">
Perhaps it was rot  
perhaps sabotage dressed gray  
either way has style
</Limerick>

<Limerick type="note" title="If the page loops back" icon="book">
If the page loops back  
you have been judged by the map  
and the map has lies
</Limerick>
