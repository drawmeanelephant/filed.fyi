## Haikus

<Limerick type="note" title="No owner present" icon="archive">
No owner present  
inventory calls that calm now  
I call it missing
</Limerick>

<Limerick type="note" title="Low touch, says the sheet" icon="file-question">
Low touch, says the sheet  
meaning no one cared enough  
to touch this at all
</Limerick>

<Limerick type="note" title="Ownership failure" icon="bug">
Ownership failure  
washed and folded into a  
stability claim
</Limerick>

<Limerick type="note" title="Unattended files" icon="stamp">
Unattended files  
stand very still on the shelf now  
thus become mature
</Limerick>

<Limerick type="note" title="Reinterpretation" icon="layers">
Reinterpretation  
is neglect learning table  
manners in a tie
</Limerick>

<Limerick type="note" title="No one raised a hand" icon="ghost">
No one raised a hand  
therefore the room names it peace  
that old office spell
</Limerick>

<Limerick type="note" title="Inventory fog" icon="sparkles">
Inventory fog  
softens every jagged edge  
except the real one
</Limerick>

<Limerick type="note" title="Touchless governance" icon="binary">
Touchless governance  
makes abandonment look like  
excellent posture
</Limerick>

<Limerick type="note" title="Some losses sit still" icon="book">
Some losses sit still  
long enough to be called wise  
that does not help them
</Limerick>

<Limerick type="note" title="When an orphan file" icon="telescope">
When an orphan file  
gets praised for its discipline  
check who left the room
</Limerick>
