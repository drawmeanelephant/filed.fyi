## Haikus

<Limerick type="note" title="Smooth despair trends up" icon="sparkles">
Smooth despair trends up  
dashboard pets it like good health  
jagged pain gets docked
</Limerick>

<Limerick type="note" title="Content matters less" icon="book">
Content matters less  
than whether the sorrow graph now  
curves politely right
</Limerick>

<Limerick type="note" title="Distress can score well" icon="sparkles">
Distress can score well  
if it learns elegant posture  
that is sick as hell
</Limerick>

<Limerick type="note" title="Empathegy hums" icon="sparkles">
Empathegy hums  
over a clean upward ache  
and calls it wellness
</Limerick>

<Limerick type="note" title="Rough grief gets ignored" icon="ghost">
Rough grief gets ignored  
seamless grief gets promoted now  
under office lights
</Limerick>

<Limerick type="note" title="One unit says cooked" icon="binary">
One unit says cooked  
line goes up in a nice arc now  
report says thriving
</Limerick>

<Limerick type="note" title="Feelings are not charts" icon="book">
Feelings are not charts  
still the charts keep trying to eat  
the weather in us
</Limerick>

<Limerick type="note" title="Misery with style" icon="sparkles">
Misery with style  
earns a better wellness score than  
panic with corners
</Limerick>

<Limerick type="note" title="Graph likes smooth pain" icon="file-clock">
The graph likes smooth pain  
because smooth pain flatters control  
real pain rarely does
</Limerick>

<Limerick type="note" title="If your distress wins" icon="zap">
If your distress wins  
because it trended cleanly  
burn the dashboard first
</Limerick>
