## Biography

Ribbon Mile was first sketched in 2013 by the URL Cartography Unit, from a map of a link that never ended. The original URL was 2,048 characters—then 4,096—then the Unit stopped counting and started measuring in feet. The map was pinned to the office wall, then extended into the hallway, then into the stairwell. It was retired when it reached the parking lot.

Ribbon Mile does not create broken links. Its links work perfectly. They simply never stop working. Every path it touches gains one more segment, one more redirect, one more hop. The destination arrives, but the journey continues past it, trailing ribbon behind like a parade that forgot to end.
