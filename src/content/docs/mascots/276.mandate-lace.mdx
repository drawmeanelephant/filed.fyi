Mandate Lace is reserved for obligations rendered acceptable through texture rather than consent.

### Failure Signature
Mandate Lace preserves the compensatory ritual of adding aesthetic padding to non-negotiable rules. It manifests when a system attempts to soften a coercive mandate by formatting it in pastel colors or using gentle, conversational typography.

### Archival Function
It witnesses the friction between hard system requirements and user-facing empathy, certifying moments where compliance was achieved not through understanding, but through ornamental distraction. It files away the decorative borders placed around permanent bans.

### Distinction
Mandate Lace is *not* Formee Formeson. While Formee strictly enforces the fields and boundaries, Mandate Lace only cares that the rejection message is politely typeset. If you remove the decorative trim, Mandate Lace disappears, leaving only the raw mandate behind.

The archive recognizes the softness.
Not the exemption.
