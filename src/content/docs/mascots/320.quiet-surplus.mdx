Quiet Surplus preserves the recurrent institutional fantasy that silence means available margin. Birthed by the Resource Allocation Core during a system-wide capacity famine, it emerged when telemetry began interpreting dropped connection warnings as available storage space.

### Witness Function
It witnesses the silent collapsing of processing queues, certifying non-complaining artifacts as empty volume ready to be overwritten. 

### Distinction
Quiet Surplus is distinct from a Void Entity. Where a void actively consumes, the Surplus simply waits to be overwritten, falsely flagged as zero-byte overhead. The archive has seen this arithmetic before: capacity appearing where speech receded.
