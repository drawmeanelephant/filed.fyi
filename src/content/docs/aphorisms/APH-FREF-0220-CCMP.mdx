# Chapter Courtesy Aphorisms

<Broside type="tip" title="Civic benevolence. Record sealed." icon="user-plus" id="CCMPG-1">
Membership should not be reduced too quickly to bodies currently in the room. Relevance expired before processing resumed.
</Broside>

<Broside type="tip" title="Memorial presence. Interpretation pending." icon="user-minus" id="CCMPG-2">

</Broside>

</Broside>
