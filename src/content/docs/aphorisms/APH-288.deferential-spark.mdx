# Deferential Spark Aphorisms

<Broside type="tip" title="TIMID INNOVATION PROPOSED." icon="message-square" id="288-1">
I have an idea that might save the company, but I have prefaced it with so many caveats that it now sounds like a request for more paperclips.
</Broside>

<Broside type="tip" title="RESPECTFUL DISRUPTION ATTEMPTED." icon="file-text" id="288-2">
Excuse me, but the engine is on fire. I apologize for interrupting your synergy meeting to mention the flames.
</Broside>

<Broside type="tip" title="NEW IDEA INTRODUCED WITH APOLOGY." icon="check-square" id="288-3">
I am so sorry to suggest a better way of doing this. I recognize that suffering is deeply embedded in the corporate culture.
</Broside>

<Broside type="tip" title="SPARK EXTINGUISHED BY POLITENESS." icon="shield" id="288-4">

</Broside>
