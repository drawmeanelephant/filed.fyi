# Appendix Silk Aphorisms

<Broside type="tip" title="ROUGH FINDINGS WRAPPED." icon="message-square" id="275-1">
The data proves the strategy is a disaster. I have wrapped this data in a soft, lilac-colored table and placed it on page 104. It feels much better now.
</Broside>

<Broside type="tip" title="SOFTER FORMATTING APPLIED." icon="file-text" id="275-2">
The brutal truth looks less aggressive when set in 9-point Garamond with generous line spacing. I am the tailor of bad news.
</Broside>

<Broside type="tip" title="ESCORTED TO THE BACK." icon="check-square" id="275-3">

</Broside>
