# Placeholder Witness Aphorisms

<Broside type="note" title="INTENTIONAL ABSENCE" icon="user-x" id="223-1">
We place a heavy brass paperweight on the empty chair. The weight ensures the absence is recorded as a structural requirement rather than a staffing failure.
</Broside>

<Broside type="tip" title="BLANK LANYARD" icon="tag" id="223-2">
The lanyard is printed and hung by the door. We do not write a name on it, but we straighten it daily so the vacancy looks highly organized.
</Broside>

<Broside type="caution" title="RESERVED PODIUM" icon="mic" id="223-3">
We regularly dust the empty podium. If we allow dust to settle, the committee might realize nobody is actually scheduled to speak for the incident.
</Broside>

<Broside type="tip" title="SPATIAL GEOMETRY" icon="layout" id="223-4">
The room requires three witnesses to proceed. We place a coat over the third chair and nod to it before every major decision.
</Broside>
