# Scheduled Exception Aphorisms

<Broside type="tip" title="Scheduled exception. Jurisdiction shifted." icon="alert-circle" id="0385-SED-1">

</Broside>

<Broside type="tip" title="Maintenance window. Status acknowledged." icon="help-circle" id="0385-SED-2">

</Broside>
