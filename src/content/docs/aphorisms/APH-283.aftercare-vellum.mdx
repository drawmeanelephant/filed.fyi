# Aftercare Vellum Aphorisms

<Broside type="tip" title="POST-INCIDENT TENDERNESS APPLIED." icon="message-square" id="283-1">

</Broside>

<Broside type="tip" title="ELEGANT FOLLOW-UP DELIVERED." icon="file-text" id="283-2">

</Broside>
