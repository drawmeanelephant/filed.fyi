# Caveat Snowglobe Aphorisms

<Broside type="tip" title="CAVEAT VISIBLY PRESERVED." icon="message-square" id="323-1">
The fatal flaw in the plan is placed inside this beautiful glass sphere. We shake it, watch the warnings swirl like snow, and then ignore it completely.
</Broside>

<Broside type="tip" title="CAUTION CLAIMED. DECISION MADE." icon="file-text" id="323-2">
We have thoroughly documented the reasons this will fail. Having achieved perfect transparency, we will now proceed exactly as planned.
</Broside>

<Broside type="tip" title="WARNING SEALED DECORATIVELY." icon="check-square" id="323-3">
The risk assessment is a lovely centerpiece on the boardroom table. Nobody is allowed to open it. It is purely ornamental risk.
</Broside>

<Broside type="tip" title="FLAW KEPT UNDER GLASS." icon="shield" id="323-4">
The red flags are enclosed in a vacuum. We can see them waving frantically, but we cannot hear the alarm. The glass is very thick.
</Broside>
