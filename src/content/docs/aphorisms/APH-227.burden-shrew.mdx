# Burden Shrew Aphorisms

<Broside type="tip" title="UNCOUNTED EFFORT INGESTED." icon="message-square" id="227-1">
I carry the weight of the work that doesn't fit in the Jira ticket. It is heavier than the ticket itself.
</Broside>

<Broside type="tip" title="CAPACITY INVISIBLE." icon="file-text" id="227-2">
If I drop this undocumented load, the system halts. If I carry it, productivity appears effortless. My exhaustion is an architectural requirement.
</Broside>

<Broside type="tip" title="SHADOW METRICS UPDATED." icon="check-square" id="227-3">
The burndown chart is empty. My back is broken. The sprint was a resounding success according to the stakeholders.
</Broside>

<Broside type="tip" title="RESILIENCE AS A SERVICE." icon="shield" id="227-4">
They call it 'systemic resilience.' I call it 'scurrying very fast in the dark.' We are both describing the same fear.
</Broside>
