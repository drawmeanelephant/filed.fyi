# Burden Shrew Aphorisms

<Broside type="note" title="RESIDUAL APPENDIX" icon="folder" id="227-1">
We file the extra weight in the unread appendix because including it in the main folder would break the spine of the executive summary.
</Broside>

<Broside type="tip" title="HYGIENE SWEEP" icon="trash" id="227-2">

</Broside>

<Broside type="caution" title="UNCOUNTED LOAD" icon="archive" id="227-3">

</Broside>

<Broside type="tip" title="SHADOW PERSISTENCE" icon="moon" id="227-4">

</Broside>
