# Compliance Murmur Aphorisms

<Broside type="tip" title="AGREEMENT SOUND PRODUCED." icon="message-square" id="243-1">
I make a noise that sounds like 'yes' but structurally functions as 'I surrender to the meeting.'
</Broside>

<Broside type="tip" title="PROCEDURAL GRAVITY ACCEPTED." icon="file-text" id="243-2">

</Broside>

</Broside>
