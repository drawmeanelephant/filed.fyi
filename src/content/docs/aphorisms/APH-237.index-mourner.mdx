# Index Mourner Aphorisms

<Broside type="tip" title="DRIFT ANTICIPATED." icon="message-square" id="237-1">
I weep for the links that will break tomorrow. The index is perfectly mapped today, which means the decay has already begun.
</Broside>

<Broside type="tip" title="FUTURE GRIEF LOGGED." icon="file-text" id="237-2">
You search for a document that was moved three reorganizations ago. I hold the 404 error like a folded flag.
</Broside>

<Broside type="tip" title="RESIDUE CATALOGED." icon="check-square" id="237-3">
Tags without files. Files without categories. I am the caretaker of the orphans left behind when the taxonomy shifted in the night.
</Broside>

<Broside type="tip" title="NAVIGATION FAILURE AVERTED PRE-POSTHUMOUSLY." icon="shield" id="237-4">
The breadcrumbs were eaten by the migration script. I mourn the paths that now lead only back to themselves.
</Broside>
