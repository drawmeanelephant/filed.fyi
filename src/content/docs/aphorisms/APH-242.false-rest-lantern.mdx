# False Rest Lantern Aphorisms

<Broside type="tip" title="SILENCE CLASSIFIED AS REST." icon="message-square" id="242-1">
The system has stopped screaming because it lacks the energy to draw breath. I illuminate this collapse with a soothing amber glow. We call it 'downtime.'
</Broside>

<Broside type="tip" title="DEPLETION DISGUISED." icon="file-text" id="242-2">

</Broside>

<Broside type="tip" title="RECOVERY ILLUSION SECURED." icon="check-square" id="242-3">
The weekend is just a brief pause in the beating. I hold the light so you can temporarily mistake the pause for healing.
</Broside>

<Broside type="tip" title="NOISE SUPPRESSION ACTIVE." icon="shield" id="242-4">

</Broside>
