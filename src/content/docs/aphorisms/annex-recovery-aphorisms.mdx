# Annex Recovery Aphorisms

<Broside type="tip" title="Partial salvage. Status acknowledged." icon="refresh-cw" id="ANX-REC-1">
Restore enough interpretive structure to prevent the archive from lying more neatly than the event occurred. The form remained intact. The situation did not.
</Broside>

<Broside type="tip" title="Not full event fidelity. Boundary active." icon="anchor" id="ANX-REC-2">
Annex Recovery is a doctrine of partial salvage; it reconstructs interpretive pressure, not full event fidelity. Meaning adjusted around the winning selector.
</Broside>

<Broside type="tip" title="Foundational Rule. Continuity retained." icon="volume-x" id="ANX-REC-3">
Some things survive only because they were inconvenient enough to be preserved somewhere quieter. Relief remained outside scope.
</Broside>
