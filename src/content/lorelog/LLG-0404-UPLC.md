## Incident Summary

During method validation for Compound 17-F in plasma, urine, and brain homogenate, a peak at RT = 3.7 min appeared in:

- All calibration standards
- All quality control samples  
- All matrix blanks
- All solvent blanks
- The system suitability injection performed before the sequence
- A sequence run the following Tuesday after a full system flush

The peak was not the analyte (RT = 6.2 min). It integrated consistently at ~340 counts. It did not interfere with quantification. The method was validated and accepted.

## Kindy's Filing Note

> "I have looked at this peak for a long time now.  
> It was there before we started. It was there after we cleaned everything.  
> It is in the blank. The blank is supposed to be nothing.  
> I have decided the peak is not a problem. I have also decided I do not fully believe that.  
> I have logged both positions. I understand this is not standard.  
> I feel the peak understands."

## Resolution: VOID

The peak was classified as a **system background artifact** and excluded from the quantification window. It appears in every run to this day. It has never been identified. The instrument service report lists it as **"within expected system noise."**

The instrument service engineer did not look at it directly.
